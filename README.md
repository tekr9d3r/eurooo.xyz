# Eurooo.xyz

A simple, high-performance dashboard for tracking and accessing Euro stablecoin yield opportunities. 

## âš¡ The Project
**Eurooo.xyz** is a pure Web3 frontend that aggregates real-time data for EUR stablecoins. It eliminates the need for a centralized backend by communicating directly with the blockchain.

* **No Database:** All data (APY, TVL, Balances) is pulled live from RPCs.
* **Non-Custodial:** Deposits and withdrawals happen directly between your wallet and the protocols.
* **Privacy-First:** No accounts, no tracking, just wallet-based interaction.
